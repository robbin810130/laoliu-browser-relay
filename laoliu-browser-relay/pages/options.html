快速上手
  1. 确保 Relay Server 已启动（默认端口 9236）
  2. 扩展会自动连接 — 图标上出现绿点即表示就绪
  3. 通过 MCP 客户端（WorkBuddy / Claude Desktop / Cursor 等）发送浏览器指令即可
工作原理
自动中继
中继默认启用，Relay Server 运行时自动连接。使用上方开关可暂停。Agent 打开的标签页会归入 "LaoLiu Agent" 标签组。
自动重连
连接断开后，扩展会自动重新连接，无需手动干预。
标签安全
你现有的标签页受保护 — Agent 只能关闭它自己打开的标签页，个人标签页不受影响。
状态指示
已连接 — Agent 可以控制你的标签页 正在连接 / 重连到 Relay Server 中继已禁用 — 无指示灯
故障排查
一直显示黄点（连接中）
Relay Server 不可达。请确认 Relay Server 已启动，默认端口 9236。
页面无法操控
Chrome 内部页面（chrome://）无法被控制，请导航到普通网页。
高级设置
标签组行为
禁用中继时关闭 "LaoLiu Agent" 标签组
控制端口
仅在使用了自定义端口时需要修改。
默认: 9234. Relay 端口 = 控制端口 + 2.
